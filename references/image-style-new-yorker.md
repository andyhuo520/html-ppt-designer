# 纽约客风格 (New Yorker Style) 配图提示词指南

## 风格定义

灵感来自《纽约客》杂志的经典插画风格。
特点：黑白为主、红色点缀、手绘线条感、知识分子幽默、叙事性构图。
适合：深度文章、评论性内容、文化/商业/科技话题、高端品牌传播。


## 配色方案（极简三色）

| 颜色 | 色值 | 用途 | 占比 |
|------|------|------|------|
| 黑色线条 | #1A1A1A | 主体线描、人物轮廓、文字 | 30-40% |
| 白色/米白 | #FAFAF5 / #FFFFFF | 背景、留白 | 40-50% |
| 红色点缀 | #C0392B / #E63946 | 关键强调（仅1-2处） | 5-10% |
| 灰色渲染 | #D0D0D0 | 阴影、层次、氛围 | 10-15% |

**红色使用规则**：
- 极度克制，每幅图最多 1-2 个红色元素
- 用于：领带、书脊、按钮、标记、杯中液体等
- 红色即视觉焦点，引导观者注意力

## 线条风格

### 基本特征
- **手绘感**：不是完美的电脑线条，有轻微的不规则
- **粗细变化**：轮廓线略粗（2-3px），细节线细（0.5-1px）
- **墨水质感**：类似钢笔/毛笔墨水的感觉
- **留白大胆**：不需要填满所有区域

### 关键术语
```plaintext
- "ink line drawing" — 墨水线描
- "pen and ink illustration" — 钢笔插画
- "editorial cartoon line work" — 社论漫画线条
- "hand-drawn feel with varying line weight" — 手绘感粗细变化
- "cross-hatching for shadows" — 交叉排线阴影
```

---

## 人物风格

### 基本原则
- **写实但简化**：不是卡通Q版，也不是照片写实
- **表情生动**：通过面部表情和肢体语言讲故事
- **着装讲究**：西装、衬衫、眼镜等知识分子形象
- **比例略夸张**：头部稍大、手势放大（社论漫画手法）

### 人物类型

| 类型 | 特征 | 常见元素 |
|------|------|---------|
| 知识分子 | 戴眼镜、手持书本 | 书架、咖啡、报纸 |
| 商务人士 | 西装革履、公文包 | 办公室、会议桌 |
| 科技工作者 | 面对屏幕、键盘 | 代码、服务器、数据 |
| 观察者 | 旁观姿态、若有所思 | 窗户、城市景观 |

## 构图方式

### 叙事性构图（核心）
纽约客风格的灵魂是"用一幅图讲一个故事"。

### 常见构图模式

**模式 A - 场景切片**
```
一个完整的场景，观者像透过窗户观看
人物在场景中自然活动
环境细节丰富但不杂乱
```

**模式 B - 概念隐喻**
```
用视觉隐喻表达抽象概念
例：人物站在巨大的键盘上 = 被技术控制
例：数据像瀑布一样倾泻 = 信息过载
```

**模式 C - 人物特写**
```
单个或少量人物的特写
重点在表情和姿态
背景简化或留白
```

**模式 D - 全景俯瞰**
```
鸟瞰视角展示复杂系统
多个小人物在系统中活动
适合展示架构、流程、生态
```

## 场景化示例

### 技术主题
```plaintext
New Yorker magazine style illustration, black and white with red accent.

Scene: A developer sitting at a desk surrounded by multiple floating screens and tangled cables. The screens show various AI chatbot interfaces all talking at once. The developer looks overwhelmed but amused, holding a single red coffee mug (the only color in the scene).

Style: Sophisticated ink line drawing, editorial cartoon aesthetic, hand-drawn feel, cross-hatching for shadows, New Yorker editorial quality. Intellectual humor. Black and white with only the coffee mug in red (#C0392B).
```

### 商业主题
```plaintext
New Yorker magazine style illustration, black and white with red accent.

Scene: A boardroom meeting where executives sit around a long table. At the head of the table, instead of a person, sits a large robot/AI entity in a business suit. Everyone else takes notes diligently. One executive's red tie (only color) draws attention.

Style: Pen and ink editorial illustration, sophisticated line work, subtle humor, minimal gray wash for depth, New Yorker magazine quality. No text.
```

### 概念隐喻
```plaintext
New Yorker magazine style illustration, black and white with red accent.

Scene: A person climbing an impossibly tall stack of books that reaches into the clouds. At the very top, a small red flag marks the summit. Below, tiny figures look up in wonder. The books have visible spines with no text.

Style: Ink line drawing with gray wash, editorial cartoon proportions, hand-drawn quality, narrative composition, intellectual mood. Only the flag is red (#E63946).
```

## 与其他风格对比

| 维度 | 纽约客 | 红黑白科技 | 卡通 2.5D |
|------|--------|-----------|----------|
| 视角 | 自由/叙事性 | 等距 2.5D | 等距 2.5D |
| 线条 | 手绘墨水感 | 精确几何线 | 圆润柔和 |
| 人物 | 写实略夸张 | 黑色剪影 | Q版卡通 |
| 颜色 | 黑白+红点缀 | 红黑白 | 多彩和谐 |
| 情绪 | 知性、幽默 | 专业、理性 | 友好、乐观 |
| 叙事 | 强叙事性 | 信息图表 | 场景展示 |
| 适用 | 深度文章、评论 | 技术文章、架构 | 教育、营销 |

## 适用场景

✅ **适合**：
- 深度分析文章配图
- 科技/商业评论
- 文化/社会话题
- 高端品牌传播
- 思想性/评论性内容
- 知识型演示文稿

❌ **不适合**：
- 儿童教育内容
- 轻松娱乐内容
- 纯数据/图表展示
- 需要多彩视觉冲击的场景
