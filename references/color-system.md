## 二、色彩心理学映射（Eva Heller + Itten 理论）

### 色相-情感映射

| 色相范围 (H) | 色彩名称 | 心理维度 | 情感映射 |
|-------------|---------|---------|---------|
| 0-15 | 红色 | 高温度 x 高唤醒 x 高支配 | 紧急、激情、危险、力量、爱 |
| 15-45 | 橙色 | 高温度 x 高唤醒 x 中支配 | 活力、乐观、创造、热情 |
| 45-65 | 黄色 | 高温度 x 中唤醒 x 低支配 | 快乐、注意、警示、智慧 |
| 65-120 | 黄绿-绿色 | 中温度 x 低唤醒 x 低支配 | 成长、自然、平衡、健康 |
| 120-180 | 青绿-青色 | 低温度 x 低唤醒 x 中支配 | 清新、科技、未来、冷静 |
| 180-240 | 蓝色 | 低温度 x 低唤醒 x 中支配 | 信任、稳定、专业、深邃 |
| 240-280 | 蓝紫-紫色 | 中温度 x 中唤醒 x 高支配 | 神秘、奢华、创意、灵性 |
| 280-330 | 品红-粉色 | 中温度 x 中唤醒 x 低支配 | 浪漫、温柔、女性化、创新 |
| N/A (S<10%) | 灰度 | 中性 x 低唤醒 x 中支配 | 中立、正式、精致、永恒 |

### 饱和度-情感强度

| 饱和度范围 | 心理效果 | 适用场景 |
|-----------|---------|---------|
| S > 80% | 强烈、刺激、醒目 | CTA 按钮、警示、创意 |
| S 50-80% | 活力、清晰、专业 | 品牌主色、图标、标题 |
| S 30-50% | 柔和、温和、可信 | 背景色块、辅助色 |
| S 10-30% | 安静、雅致、复古 | 大面积背景、极简设计 |
| S < 10% | 中性、无情感倾向 | 文本、边框、分隔线 |

## 四、和谐色板构建算法

### 互补色 (Complementary)
```
Primary H -> Complement H = Primary H + 180
应用：标题 vs 强调色，高对比引导视线
注意：面积比至少 7:3，小面积用互补色
```

### 分裂互补 (Split-Complementary)
```
Primary H -> Split1 H = Primary H + 150
           -> Split2 H = Primary H + 210
应用：平衡对比与和谐，适合三色板
```

### 三角和谐 (Triadic)
```
Primary H -> Triad1 H = Primary H + 120
           -> Triad2 H = Primary H + 240
应用：活力多彩场景，需控制饱和度
```

### 单色渐变 (Monochromatic)
```
固定 H，变化 S 和 V：
- 最深：H, S:80%, V:30% -> 强调
- 中间：H, S:60%, V:55% -> 主色
- 浅色：H, S:30%, V:85% -> 背景色块
- 最浅：H, S:10%, V:95% -> 页面背景
```
